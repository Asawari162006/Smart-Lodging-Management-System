 Smart Lodging Management System

A full-stack web application designed to streamline hotel operations by managing room bookings, guest records, billing, and reservations. The system also incorporates intelligent features such as **dynamic pricing**, **payment gateway simulation**, an **AI chatbot**, and **customer sentiment analysis** to enhance operational efficiency and user experience.

---



Core Features
- User Authentication & Role-Based Access
- Room Booking & Reservation Management
- Guest Check-in / Check-out Management
- Billing & Invoice Generation
- Customer Management
- Room Availability Tracking

 Smart Features
- Dynamic Pricing based on booking conditions
- Payment Gateway Simulation for booking workflow
- AI Chatbot for customer assistance and booking queries
- Customer Sentiment Analysis from feedback/reviews

Installation

 1. Clone the repository

```bash
git clone https://github.com/Asawari162006/Smart-Lodging-Management-System.git
```

---

 2. Navigate to the project

```bash
cd Smart-Lodging-Management-System
```

---

 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

4. Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

 5. Configure Database

- Create a MySQL database.
- Import the SQL file located in the **database** folder.

Example:

```
database/database.sql
```

---

 6. Start Backend

```bash
npm start
```

---

 7. Start Frontend

```bash
cd ../frontend
npm start
```

The application will be available at:

```
http://localhost:3000
```
